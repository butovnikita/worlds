# 🔄 Lorebook Keyword Translator (RU) - SillyTavern Extension

Расширение для [SillyTavern](https://github.com/SillyTavern/SillyTavern), которое переводит **Keywords** (ключевые слова) в Lorebooks на русский и другие языки.

## ✨ Возможности

- 🌐 Перевод всех ключевых слов в Lorebook через онлайн-переводчик
- 🔤 Поддержка множества исходных языков (English, Deutsch, Français, Español, Italiano, 日本語, 中文, 한국어)
- 🇷🇺 Целевой язык по умолчанию - русский
- 🔄 Перевод как основных, так и вторичных ключевых слов
- 📊 Отображение прогресса перевода в реальном времени
- ⚡ Бесплатный API без необходимости получения ключа

## 📦 Установка

### Способ 1: Через папку расширений

1. Скачайте или клонируйте эту папку:
   ```bash
   git clone <repository-url>
   ```

2. Скопируйте папку `sillytavern-keyword-translator` в директорию расширений SillyTavern:
   ```
   SillyTavern/public/scripts/extensions/sillytavern-keyword-translator/
   ```

3. Перезапустите SillyTavern

### Способ 2: Ручная установка

1. Создайте папку `sillytavern-keyword-translator` в:
   ```
   SillyTavern/public/scripts/extensions/
   ```

2. Скопируйте в неё следующие файлы:
   - `manifest.json`
   - `index.js`
   - `styles.css`

3. Перезапустите SillyTavern

## 🚀 Использование

1. Откройте SillyTavern
2. Перейдите в **Настройки** → **Расширения** (Extensions)
3. Найдите секцию **"🔄 Переводчик Keywords Lorebook"**
4. Выберите **исходный язык** (язык оригинальных ключевых слов)
5. Выберите **целевой язык** (по умолчанию - Русский)
6. Выберите нужный **Lorebook** из списка
7. Нажмите кнопку **"🌐 Перевести Keywords"**

## 🔧 Как это работает

1. Расширение загружает выбранный Lorebook через API SillyTavern
2. Для каждой записи в Lorebook:
   - Извлекаются все ключевые слова (`key` и `keysecondary`)
   - Каждое ключевое слово отправляется на перевод через MyMemory API
   - Переведённые слова заменяют оригинальные
3. Обновлённый Lorebook сохраняется обратно

## 📋 API Translation

Расширение использует **MyMemory Translation API**:
- Бесплатный сервис
- Не требует API ключа
- Лимит: ~5000 слов в день для незарегистрированных пользователей
- Поддержка множества языковых пар

## ⚠️ Известные ограничения

- Перевод специфических терминов может быть неточным
- Длинные ключевые слова могут быть разбиты на части API
- При большом количестве ключевых слов перевод может занять время
- Рекомендуется делать резервную копию Lorebook перед переводом

## 🛠️ Для разработчиков

### Структура файлов

```
sillytavern-keyword-translator/
├── manifest.json    # Метаданные расширения
├── index.js        # Основная логика расширения
├── styles.css      # Стили UI
└── README.md       # Эта документация
```

### Основные функции

- `translateText()` - Перевод одного текста через API
- `translateLorebook()` - Перевод всего Lorebook
- `translateEntryKeywords()` - Перевод ключевых слов одной записи
- `injectUI()` - Внедрение пользовательского интерфейса
